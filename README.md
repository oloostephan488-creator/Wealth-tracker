# WealthTrack Live | Portfolio Manager 🦁📈

A high-performance, real-time cryptocurrency and asset tracking dashboard. This project was built to demonstrate proficiency in **Asynchronous JavaScript**, **REST API integration**, and **Dynamic UI rendering**.

## 🛠️ Tech Stack
- **HTML5 & CSS3**: Custom "Fintech-Dark" theme with glassmorphism elements.
- **Vanilla JavaScript**: Used for all logic (no heavy frameworks).
- **CoinGecko API**: Real-time market data fetching.
- **Acode Mobile IDE**: Entirely developed on a mobile environment.

## 🧠 Key Engineering Features
- **Real-Time Data Fetching**: Utilizes `async/await` and the `fetch()` API to pull live global market rates every 60 seconds.
- **Dynamic State Management**: A local wallet array that updates the UI automatically without page refreshes.
- **Custom Search Filter**: A real-time search engine that filters assets by name or ticker using JavaScript `.filter()`.
- **Responsive Mobile-First Design**: Optimized for high-speed performance on mobile devices.
